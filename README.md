# Admin CRM

CRM‑панель для управления пользователями и заказами. Проект, демонстрирующий навыки frontend‑разработки на Vue 3 + TypeScript.

# Стек

1) Vue 3 (Composition API, <script setup>)
2) TypeScript
3) Vite
4) Pinia (state management)
5) Vue Router
6) Axios
7) SCSS (CSS‑переменные, миксины, адаптив)
8) Chart.js (графики)
9) json‑server (mock API)
10) Vitest (unit‑тесты)

# Основные возможности

1) Авторизация (хранилище токена в localStorage, защита маршрутов)
2) CRUD для пользователей и заказов (создание, редактирование, удаление)
3) Пагинация и фильтрация (поиск по тексту, статус заказов)
4) Дашборд с графиками: общая статистика, столбчатая диаграмма по статусам, линейный график выручки по дням
5) Переключение светлой / тёмной темы (сохранение выбора)
6) Адаптивный дизайн (мобильная версия)
7) Кастомные уведомления (тосты)
8) Unit‑тесты (пример для стора)

# Запуск проекта

Установка зависимостей:

npm install

Запуск одной командой:

npm run dev:all

Тестовые учётные данные:

    Логин: admin@example.com / пароль: admin123

    Или user@example.com / user123
